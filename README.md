# 発表会用タイマー ＆ 音源ポン出しツール (Presentation Timer & Sound Board)

発表会やイベントの進行・ワンオペ操作をスムーズにするために開発した、ストップウォッチ機能と効果音（SE）ポン出し機能が一体になったWebツールです。
効果音を鳴らした瞬間のタイムスタンプを自動で記録する機能を備えています。

## 🚀 デモ（今すぐ使う）
GitHub Pagesを利用して公開しています。以下のURLからスマホやPCのブラウザで直接使用できます。

https://aid-sparkle.github.io/Gemini-timer/

---

## ✨ 主な機能

1. **高精度ストップウォッチ**
   * 発表時間をミリ秒（0.1秒単位）までリアルタイムに計測します。
   * スタート / ストップ / 再開 / リセットに対応しています。
2. **4種類の効果音ポン出し（SE）**
   * 外部の音源ファイル不要（Web Audio APIによる自動音声生成）。
   * ネット環境が不安定な会場でも、遅延なく音が鳴ります。
3. **タイムスタンプ自動記録（ログ機能）**
   * 効果音を鳴らした「その瞬間」のタイマー時間を自動で記録。
   * 最新の履歴が一番上にリストアップされ、発表のタイムキープや振り返りに役立ちます。
   * タイマーのリセットと連動してログもクリアされます。

## ⌨️ 操作方法

### PC（キーボードショートカット対応）
マウスでのクリック操作に加え、キーボードの数字キーで素早く音を鳴らすことができます。

| キー | 割り当てられる音 | 主な用途 |
| :---: | :--- | :--- |
| **1** | ピンポン | 正解の合図、注目を集めたいとき |
| **2** | ブブー | 不正解の合図、警告・タイムアップ |
| **3** | ジャジャーン | ゲスト登場、結果発表、強調したい場面 |
| **4** | 拍手 | 発表終了時、賞賛、歓声の代わり |

### スマートフォン・タブレット
* 画面上の各ボタンをタップして操作します。
* **⚠️ 注意点:** スマホブラウザの音声誤再生防止の仕様上、**ページを開いた直後に一度だけ、画面のどこかをタップする（またはテストでSEボタンを押す）** 必要があります。これを行うことで、以降タイマー動作中もスムーズに音が鳴るようになります。

## 📦 ローカルでの実行方法
インターネットがない環境でも、リポジトリから `index.html` をローカルPCにダウンロード（またはクローン）し、ダブルクリックでブラウザで開くだけで完全に動作します。
